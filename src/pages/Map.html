Map: Making your Objects Iterable

The Map() contructor is a very cool way to create an iterable data structure using key value pairs.

One problem I run into with using Svelte is that the #each template block can only take an array, whereas a lot of my data is in objects. Map() seems like a pretty cool alternative when I need object data to be iterable. The way to do that automatically is by using the entries() method. That is way better than runing all of my data through a loop to rewrite it as an array!
